Selection Sort

Selection Sort is a simple way to arrange numbers or items in order. Imagine you have a list of numbers, and you want to sort them from the smallest to the biggest. Here's how it works: 1. Start at the beginning of the list. 2. Look through all the numbers to find the smallest one. 3. Swap that smallest number with the first number in the list. 4. Now, ignore the first number (since it's already in the right place) and repeat the process for the remaining numbers. 5. Keep repeating until all the numbers are in the right order. For example, if you have the list [4, 2, 5, 1], you would first find 1 (the smallest) and swap it with 4, then find 2 and swap it with the second number, and so on.
