Bubble Sort

Bubble Sort is another simple way to sort numbers or items. It compares adjacent numbers and swaps them if they are in the wrong order. Here's how it works: 1. Start at the beginning of the list. 2. Compare the first number with the second number. If the first is larger, swap them. 3. Move to the next pair of numbers and repeat the process until you reach the end of the list. 4. After one pass through the list, the largest number will be at the end. 5. Now, ignore the last number (since itâ€™s already in the right place) and repeat the process for the remaining numbers. 6. Keep repeating this process until no more swaps are needed. For example, if you have the list [4, 2, 5, 1], you would first compare 4 and 2, swap them to get [2, 4, 5, 1], and continue comparing and swapping until the list is sorted. Though bubble sort is easy to understand, it can be slow for large lists since it requires many comparisons and swaps.
